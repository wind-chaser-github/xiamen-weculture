# 厦门旅行指南 - 文旅智慧体验小程序

厦门旅行指南是一个全方位的文旅服务平台，旨在提升游客在厦门的出行与互动体验。平台不仅提供行程报名、景点指南和旅行灵感分享，更融合了基于智能 AI 技术的专属打卡照生成等趣味互动功能。

## 🌟 核心功能特性

- **🤖 智能 AI 打卡生成（AI 穿越照）**：
  - 用户可上传或现场拍摄人像照片，系统自动与精选的厦门标志性历史/现代景点进行智能融合，一键生成精美的专属 AI 打卡穿越照。
  - 支持智能抠图、光影匹配与背景融合，生成独一无二的纪念照。
- **🔑 微信一键静默自动登录**：
  - 基于 `wx.login` 获取凭证并由后端自动换取 `openid` 进行唯一账号绑定。
  - **状态保持与数据隔离**：即使清空小程序客户端缓存，再次进入也能静默自动登录，找回已绑定的历史数据与个人“我的 AI 照片”列表。
- **📅 行程报名**：分类展示城市漫游、海岛路线、研学体验等活动；支持后台灵活定制报名表单字段并导出 Excel。
- **🗺️ 景点指南**：分类化导航展示景点图文指南。
- **💡 旅行灵感**：支持用户发布、管理个人的旅行精彩图文。
- **🛡️ 纯净去技术化与高安全性**：
  - 界面去除了所有关于底层技术提供商的暴露细节（如火山引擎、豆包模型等），文案统一展现为“智能 AI”。
  - 完善的 Git 敏感信息防泄露设置，自动过滤服务器秘钥、本地数据库和证书文件。

---

## 🛠️ 技术栈与运用

- **前端**：微信小程序原生开发，集成 `wx.login`、自拍及文件上传。
- **后端**：
  - 支持微信云开发模式（`BACKEND_MODE: 'cloud'`）。
  - 提供全新的独立 Node.js 轻量级 HTTP 后端（`BACKEND_MODE: 'http'`），可部署于自有 Linux 服务器或云主机，自带 JSON 数据库存储和 AI 服务对接层。
- **安全过滤**：配置了严密的 `.gitignore` 安全网罩，防止 `.pem` / `.key` 密钥和本地数据库被意外泄露推送。

## 🚀 快速启动指南

### 1. 本地/服务器后端启动
后端目录为项目根目录下的 `backend/`。

```bash
cd backend
npm install   # 安装依赖
npm start     # 启动服务
```
默认监听端口为 `3005`（可更改 PORT 环境变量），入口为 `http://127.0.0.1:3005`。

### 2. 小程序端配置
在 `miniprogram/setting/setting.js` 中配置您的后端模式与服务器地址：

```javascript
module.exports = {
	BACKEND_MODE: 'http', // 切换为 http 后端
	API_BASE_URL: 'https://tagen-preview.taowhale.com', // 替换为您的后端服务器公网域名地址
}
```

### 3. 微信一键登录与 AppSecret 配置
在部署后端时，如果需要使用微信真实的 openid 关联：
- 在运行服务器中配置环境变量 `WX_APP_SECRET` 为您小程序的 AppSecret。
- **开发与降级测试**：若不配置该环境变量，后端将自适应进入 Mock 模式，根据微信端传入的 code 模拟 openid。即使没有配置秘钥，前端开发者工具和测试真机也完全能够走通一键静默登录与专属照片隔离绑定的全部闭环逻辑。

## 📂 项目文件结构
```text
├── backend/                   # 独立 Node.js 后端服务器
│   ├── server.js              # 主服务入口，处理接口路由与静态文件服务
│   ├── db.js                  # 本地 JSON 数据存取管理器
│   ├── data/                  # 本地 JSON 数据库目录
│   └── public/                # 静态资源及用户上传图片保存目录
├── miniprogram/               # 微信小程序前端源码
│   ├── comm/biz/              # 通用业务逻辑（包含 passport_biz.js 用户管理）
│   ├── helper/                # 工具辅助函数（包含 cloud_helper.js）
│   ├── projects/culture/      # 智慧文旅项目核心业务页面
│   └── setting/setting.js     # 前端环境与 API 配置项
└── project.config.json        # 微信小程序配置文件
```
